Game Link Instructions

We are conducting a game about memory on different dataset in the link.

Before the game - At the top side of page, you can change the dataset of the game. Please make sure that you won't be disturbed during 5 minutes.

First login - You can press N to generate a random nickname, we will ask you a few questions about yourself for the first time (you can choose "I don't want to reply")

In the game - Press Space to mark the images that are shown twice. (Details in the help menu in game.)

After the game - Paste your nickname and enter your password to get the rewards. Please don't share your password with the others, otherwise, your account might be rejected!

Rewards - At least one valid game will reward you $0.1, each 4 valid games on 4 different dataset will reward you $0.4. There are up to 6 games available for each dataset ($2.5)

Valid game - We designed in-game metrics to evaluate whether you have actually concentrated on the game (valid game) or not (invalid game). Don't worry, it won't conduct invalid game because of your bad memory.
